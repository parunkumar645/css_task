HTML Terms

HTML
HyperText Markup Language - the standard markup language for documents designed to be displayed in a web browser.
HTML element
An individual component of an HTML document, consisting of an opening tag, content, and a closing tag.
HTML tag
A set of characters in a document that defines an element, including the opening and closing tags.
Angle bracket
The characters `<` and `>` used to define HTML tags.
HTML attribute
Additional information about an HTML element, defined using key-value pairs within the opening tag.
HTML comment
Text within that is not displayed on the webpage and is used for adding comments to the code.
HTML object element
An HTML element used to embed external resources such as images, audio, or videos into a document.
HTML special character
A character that has a special meaning in HTML, such as < representing the less-than sign.
